Using chatbot in Wechat

Wed 21 June 2017
By Zhijun Gao

机器人当班微信

因为想基于微信做一个问答系统，例如回答计算机辅助翻译专业考研问题等。于是探索了一下可能性。Python 有个开源聊天机器人，chatterbot，非常不错，训练也比较简单。为了和微信结合我又尝试了一个wxpy模块，试验过程记录如下。

wxpy

简介：基于itchat开发的可以控制微信网页端的模块。

网站：http://wxpy.readthedocs.io/zh/latest/

from wxpy import *
bot=Bot()

发送消息

# 给机器人自己发送消息
bot.self.send('hello wechat')
# 给文件助手发送消息
bot.file_helper.send('Hello World')

指定聊天好友

# 如需要将京燕设置为好友，则可查找后赋值
my_friend=bot.friends().search('京燕')[0]

给好友发送消息

my_friend.send('Hello, WeChat!')
# 发送图片
my_friend.send_image('my_picture.png')
# 发送视频
my_friend.send_video('my_video.mov')
# 发送文件
my_friend.send_file('my_file.zip')
# 以动态的方式发送图片
my_friend.send('@img@my_picture.png')

自动回复好友消息

@bot.register(my_friend)
def reply_my_friend(msg):
    return '您的消息已收悉: {} ({})'.format(msg.text, msg.type)

用聊天机器人自动回复

bot = Bot()
my_friend = ensure_one(bot.search('游否'))
tuling = Tuling(api_key='你申请的 API KEY')

# 使用图灵机器人自动与指定好友聊天
@bot.register(my_friend)
def reply_my_friend(msg):
    tuling.do_reply(msg)

    return 'You messege is well received:{} ({})'.format (msg.text, msg.type)
# 留言机器回复
@bot.register(hjy)
def reply_my_friend(msg):
    tuling.do_reply(msg)
# 配置图灵机器人.tuling123.com
tuling = Tuling(api_key='ee47fb929e5647caa78b3aab8a2c0fe1')

group=bot.groups().search("工程系")[0]
