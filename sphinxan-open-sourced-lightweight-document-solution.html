Sphinx：一种轻量的开源文档解决方案

软件介绍

Sphinx是用Python开发的免费文档解决方案，可以轻松制作各种用途的优美文档。最早是用于Python软件自身文档的写作，目前已经成为Python开发的软件的文档首选。随着Sphinx地不断开发，已经可以用于各类文档项目，还有不少人将其用于写作图书和学位论文等。

Sphinx优势：

  1. 免费。因为是开源的，所以无需软件授权费用，非常适合开源项目或者中小型文档开发项目。另外还可以托管在Github Pages上，连托管费用都可以省去。
  2. 输出格式丰富。除了输出位HTML （包括 HTML Help）外，还可以输出位PDF，ePUB等常见格式。
  3. 模板丰富。Sphinx有多个模板可用，而且可以用jinjia方便地开发自己的模板
  4. 交叉引用方便。可以轻松设置交叉引用，还可以自动生成索引。
  5. 方便扩展。因为是开源项目，有很多志愿者在不断开发和拓展项目

快速示例

这里将使用官方的QuickStart来示例，大体步骤分为：

  1. 安装Sphinx
  2. 创建内容
  3. 发布文档

安装Sphinx

这里用pip来安装，对pip不熟悉，可以看这个文档

  1. 在 Mac Terminal中用pip安装
$ pip install Sphinx
  1. 配置quickstart文档
$ sphinx-quickstart

运行后，会以会话的形式，进行设置：

> Root path for the documentation [.]: Desktop/ttt
> Separate source and build directories (y/n) [n]: y
> Name prefix for templates and static dir [_]: .

这样便安装并完成了Quickstart的配置。

添加内容

定义文档结构

在source文件内有自动生成的index.rst文件，在这里可以文档目录结构。

一开始为空，内容如下：

.. toctree::
   :maxdepth: 2

可以往里添加目录结构

.. toctree::
   :maxdepth: 2

   intro
   tutorial
   ...
添加内容

Sphinx接受 reStructuredText 创作的文档，使用任意文本工具创作第一个文件 intro.rst，内容：

An Introduction to Sphinx
##############

:date: 2010-10-03 10:20
.. index::
   single: script; directory


Sphinx is a tool that makes it easy to create intelligent and beautiful documentation, written by Georg Brandl and licensed under the BSD license.

发布文档

首先在Mac Terminal中浏览至文档的根目录，本例中为 Desktop/Sphinx-test

$ sphinx-build -b html sourcedir builddir

运行后，在build文件夹中便可以看到发布后的文档。

可以看到索引部分也是可以正常工作的。

总结

从上方示例可以看出，Sphinx是一种相对简单和完整的文档方案，因为其免费开源的特点，加上使用Python作为开发语言，开发人员可以轻松定制和拓展其功能，相信会受到越来越多文档团队的喜欢。

此外使用rst作为内容创作语言，只需要简单培训便可熟悉其语法规则，文档可复用性上也会有很大的提高。如果您的公司正在寻找一种轻量型文档发布方法，Sphinx觉得值得一试。
