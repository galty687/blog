My PhD Project

Fri 03 December 2010
By Zhijun

My PhD project is to work out a model which could predict the usability of technical documents (information).

Purpose

Conducting usability tests is expensive which prevents many technical writers from using usability testing as a method to improve their information design. In this project, I am proposing an automatic method to predict usability based on usability predictors or metrics.

Currently the project is still under research.
